MorphAnalysis.py
----------------

http://sampark.iiit.ac.in/hindimorph/web/restapi.php/indic/morphclient

Input in the program:
----------------------

trainfoldername = 'Training folder name'
newfoldername = 'Training folder with morphological analysis'
