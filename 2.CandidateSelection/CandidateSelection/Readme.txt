Candidate Selection:

This program involves:

Selection of annaphoras - This is done by picking pronouns with POS tag "PRP" which are at the head of the noun chuck B-NP.

Candidate Selection - For each annaphora, select all the candidates based on a set of rules, for example,

a) Candidate should be a NN or a NNP present in the current sentence or in the previous sentence relative to the annaphora.
b) Gender and number of the annaphora and antecedent should match
